# Assets de la Aplicación

Esta carpeta contiene los archivos de recursos necesarios para la aplicación.

## Archivos requeridos:

### icon.png
- **Dimensiones**: 1024x1024 píxeles
- **Formato**: PNG
- **Descripción**: Icono principal de la aplicación
- **Uso**: Se muestra en la pantalla de inicio del dispositivo

### splash.png
- **Dimensiones**: 1284x2778 píxeles (recomendado)
- **Formato**: PNG
- **Descripción**: Pantalla de carga inicial
- **Uso**: Se muestra al abrir la aplicación

### adaptive-icon.png
- **Dimensiones**: 1024x1024 píxeles
- **Formato**: PNG
- **Descripción**: Icono adaptativo para Android
- **Uso**: Icono que se adapta a diferentes formas en Android

### favicon.png
- **Dimensiones**: 32x32 píxeles
- **Formato**: PNG
- **Descripción**: Favicon para la versión web
- **Uso**: Se muestra en la pestaña del navegador

## Nota importante:
Los archivos actuales son placeholders. Debes reemplazarlos con imágenes reales antes de publicar la aplicación.
